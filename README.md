# nebula_test
Получаю доступ к своим устройствам из сети в обход NAT провайдера МТС

Появилась необходимость доступа к свом домашним устройствам из любой точки, где есть сеть с доступом интернет.
После недолгих поисков в гугле наткнулся на несколько решений данной проблемы :)
Решил разобраться с проектом Nebula (Open source overlay networking)
И заработало.
Изначальная цель: связать ноут, консоль и пару одноплатников под управлением linux в единую сеть и оперативно получать информацию со всех подключеных устройств.
Почему linux? Да достал уже windows 11 со своими косяками в обновлениях, расходом батареи мобильных устройств и необоснованой сетевой активностью связаной со сбором данных :)
